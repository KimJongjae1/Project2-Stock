# 핵심 기능

- 기간 별 **주가** 그래프
    - 1주, 1달, 3개월, 6개월, 1년, 직접 설정(당일 제외)
- 기간 별 **관심도** 그래프
    - 1주, 1달, 3개월, 6개월, 1년, 직접 설정(당일 제외)
    - 관심도 기준 : 뉴스 기사 수
- 해당 기간 기업의 키워드 보여주기
    - 크롤링 기업 대상 : 2025.09.01(월) 09시 기준 코스피 상위 200개 기업(고정)
    - 뉴스 크롤링 기준 : 제목, 본문 둘 중 하나에라도 기업명 들어간 것
    - 사용자가 그래프에서 기간을 지정하여 검색을 하면 해당 기간 동안의 회사의 키워드를 보여줍니다.
    - 키워드 나열 기준 - 전체/특정 기업 트렌드 키워드
        - 전체 뉴스에서 가장 많이 나온 키워드 Top 10 (대쉬보드 나열식)
        - ~~(키워드 / 전체기사 수) 비율을 보여주기~~ (mvp 이후 고민)
- 투자 성향 파악 게임
    
    게임 진행 방식
    
    (사전 준비: 시드 - 천만원, 게임 설정 - 간격 (일, 주, 월))
    
    - 특정 기간까지 주식 차트를 보여준다. (끝나는 기간은 최대 오늘, 기업명 블라인드)
    - 특정 기간의 마지막 시점 키워드를 보여준다.
    - 투자 판단 (매수, 매도 - 금액 얼마를 살껀지)
    - 다음 간격을 보여준다. (다음 간격의 키워드, 제목 (뉴스 요약이 가능하면 요약 까지), 봉을 보여준다)
    - 다음 간격의 투자 판단 (매수와 매도를 하지 않을거면 넘어가기)
    - 끝날 때 까지 반복
    - 중간 종료 가능
    - 끝났을때 사용자 분석 보여주기
        - 자산 증가와 실현 손익을 보여준다
